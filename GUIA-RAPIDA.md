# ⚡ GUÍA RÁPIDA - PERSONALIZAR INVITACIÓN

## 🎯 EN 3 PASOS:

### PASO 1: Editar config.js
Abre `config.js` y cambia:
- ✏️ Nombre del niño
- 🎂 Edad
- 📅 Fecha (texto y formato AAAA-MM-DD)
- ⏰ Hora
- 📍 Lugar
- 📱 WhatsApp
- 🗺️ Google Maps

### PASO 2: Elegir tema
```javascript
tema: "pawpatrol"  // o "frozen", "spiderman", "princesas"
```

### PASO 3: Cambiar imagen (opcional)
```javascript
imagenTema: "tu-imagen.jpg"
```

## 📱 FORMATO WHATSAPP:
```
549 + código área + número
Ejemplo: 5491112345678
```

## 📅 FORMATO FECHA CONTADOR:
```
"2026-04-04T16:00:00"
Año-Mes-Día T Hora:Minuto:Segundo
```

## 🎨 TEMAS DISPONIBLES:
- `pawpatrol` - Azul marino
- `frozen` - Celeste/violeta
- `spiderman` - Rojo/azul
- `princesas` - Rosa/morado
- `personalizado` - Tus colores

## ✅ VERIFICACIÓN RÁPIDA:
1. ¿Guardaste config.js?
2. ¿La fecha está bien escrita?
3. ¿El número de WhatsApp es correcto?
4. ¿La imagen existe en la carpeta?

## 🚀 LISTO PARA COMPARTIR:
- Abre index.html en el navegador
- O súbelo a GitHub Pages
- Comparte el link
